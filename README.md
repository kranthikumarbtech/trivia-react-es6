# trivia-react-es6
Trivia application is written with React and ES6

1. Clone this repo
    ```
    git clone https://github.com/KranthiKumarS/trivia-react-es6.git
    ```

2. Install the dependencies
    ``` 
    yarn
    ```

3. Build the app
    ```
    gulp
    ```
4. Demo

   [Trivia](https://kranthikumars.github.io/trivia-react-es6/)
